# InTurn.AI App Overview

InTurn.AI is an internship assistant with three user-facing modes:

- InternTrack: logs internship hours, calculates remaining hours, and estimates the end date.
- MentorBridge: rewrites or evaluates workplace messages for supervisors, mentors, and coordinators.
- Report Writer: turns rough task notes into a clean daily, weekly, or monthly activity report.

Use the app for internship-related questions only. The assistant is designed to stay within scope and avoid answering unrelated coding or general advice questions.

The frontend is a Streamlit app and the backend is a FastAPI service. Chat history, reports, and configuration data are stored in MongoDB.
