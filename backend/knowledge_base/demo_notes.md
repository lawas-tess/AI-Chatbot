# Demo Notes

For a class demo, explain the chatbot in three parts:

1. Retrieval-Augmented Generation: the app searches local markdown knowledge files before answering.
2. Agentic behavior: the model can call backend tools, including the knowledge search tool and the internship status tool.
3. Safety: the router checks for non-English input, prompt injection attempts, and out-of-scope programming requests.

If asked how the project was improved, describe it as moving from a basic LLM wrapper to a guarded AI application with retrieval, tool use, and prompt-defense controls.
